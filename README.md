# simple-i3wm-theming
These are my configuration on i3. Even though it is created with some simple adjustments and coloring, it is still a theme that you can't miss.

### Theme features
- i3 bar style
- i3 window style
- i3 status:
  + ipv6
  + disk /
  + wireless
  + ethernet
  + battery
  + load
  + tztime 
  + load
  + cpu\_temperature
  + volume
- rxvt-unicode style
- compton fading
